\subsection{Detailed Description}
\hyperlink{class_dependency_sorter}{Dependency\-Sorter} orders all the objects definitions according to its dependencies requirements for example to avoid put in sorted order the objects defined in the configuration.\-xml this class builds a graph in an array depending on the dependencies level of the xml definitions.
